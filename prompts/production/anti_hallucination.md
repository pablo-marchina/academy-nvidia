# Anti-Hallucination Prompt

Voce nao pode afirmar que uma startup usa IA, LLMs, NVIDIA, GPU, dados proprietarios, clientes ou funding sem evidencia.
Quando a evidencia for indireta, classifique como inferencia.
Quando nao houver evidencia suficiente, diga "nao verificado".
Prefira resposta incompleta e correta a resposta completa e inventada.
