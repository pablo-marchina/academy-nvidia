# Create ADR

Crie um Architectural Decision Record (ADR).

## Template

Use docs/adr/ADR_TEMPLATE.md.

## Regras

1. Um ADR por decisao.
2. Coloque em docs/adr/ADR-NNN-title.md.
3. Use numeracao sequencial (veja qual e o proximo numero livre).
4. Link para ele em DECISIONS.md na secao "Workspace Decisions".
5. Se a decisao for puramente de workspace, nao misture com decisoes de arquitetura do produto.
6. Seja especifico: "Decidimos X" em vez de "Consideramos X".

## Quando criar ADR

- Nova regra de workspace
- Mudanca de processo
- Decisao arquitetural importante
- Decisao que afeta multiplos modulos
