# Projeto NVIDIA Startup AI Radar

## Objetivo
Construir uma plataforma multiagente para identificar startups brasileiras AI-native, coletar dados publicos, diagnosticar maturidade tecnica e recomendar tecnologias NVIDIA adequadas.

## Usuario final
Gerente de Startups & VCs da NVIDIA no Brasil.

## Pergunta norteadora
Como a NVIDIA pode identificar, atrair e nutrir startups brasileiras AI-native em um contexto no qual grandes labs ameacam startups que dependem apenas de wrappers de LLM?

## Entregaveis
1. Pipeline de scraping
2. Sistema multiagente com LangGraph
3. RAG NVIDIA com reranking
4. Motor de recomendacao
5. Interface web
6. Diferencial competitivo

## Criterio maximo de qualidade
O sistema deve produzir recomendacoes com evidencias, justificativa tecnica, justificativa de negocio, prioridade, complexidade e proxima acao sugerida.

## Principio do projeto
Este projeto nao e apenas uma aplicacao com IA. E um sistema para controlar a IA por meio de contexto, fontes, schemas, skills, evals, revisao humana e rastreabilidade.
