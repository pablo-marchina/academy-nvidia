# Matriz de maturidade AI-native

## Nivel 0 - Non-AI
A startup nao apresenta uso relevante de IA.

## Nivel 1 - AI-assisted
Usa IA internamente, mas nao no produto principal.

## Nivel 2 - AI-enabled
Tem funcionalidades de IA no produto, mas elas nao sao o nucleo do valor.

## Nivel 3 - AI-native
O produto depende diretamente de IA para entregar valor central.

## Nivel 4 - AI-native service
Combina IA, workflow operacional, dados proprietarios e servico especializado para entregar resultado de negocio.
