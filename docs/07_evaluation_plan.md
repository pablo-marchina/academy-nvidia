# Evaluation Plan

## What will be evaluated

- Schema adherence
- Evidence traceability
- AI-native classification quality
- Retrieval relevance
- Recommendation usefulness
- Briefing clarity

## Evaluation modes

- Unit tests for deterministic logic
- Golden examples for output comparison
- Manual review for nuanced startup assessments
- RAG metrics for retrieval quality

## Minimum bar

The project should prefer incomplete but correct outputs over complete and speculative ones.
