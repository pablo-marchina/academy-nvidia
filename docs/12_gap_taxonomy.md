# Gap Taxonomy

Os gaps tecnicos iniciais suportados pelo projeto sao:

- external_api_dependency
- high_inference_cost
- high_latency
- agent_governance_gap
- observability_gap
- model_evaluation_gap
- privacy_or_controlled_deployment_gap
- slow_data_pipeline
- heavy_tabular_processing
- voice_need
- simulation_need
- computer_vision_need
- robotics_need
- healthcare_compliance_need
- ai_cybersecurity_need
