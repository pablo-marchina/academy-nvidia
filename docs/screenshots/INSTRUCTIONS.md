# Screenshots — Final Demo

## How to Capture

1. Run the full demo script (see README — Demo Script section).
2. Use your system's screenshot tool (Print Screen, Snipping Tool, etc.).
3. Save files in `docs/screenshots/` or `docs/assets/final_demo/`.
4. Use PNG format, 1920x1080 resolution recommended.
5. Capture **after** the demo has been fully executed with real data.

## Notes

- Screenshots are **optional evidence**, not a delivery gate.
- If a screenshot cannot be captured (e.g., environment limitation), document in `docs/72_final_acceptance_evidence.md`.
- Do not include sensitive data (API keys, internal URLs) in screenshots.
- Re-capture before final delivery if the UI has changed significantly.
