# Agent Specs

## Search Planner

Define search intent, target sources, and missing evidence categories.

## Scraper Agent

Collect public pages in a policy-compliant way and preserve fetch metadata.

## Extractor Agent

Produce structured facts, evidence objects, and uncertainty markers.

## Classifier Agent

Assign AI-native maturity using the documented matrix and signal thresholds.

## Evidence Validator Agent

Reject unsupported claims and downgrade weak evidence to inference or unverified.

## NVIDIA RAG Agent

Retrieve relevant NVIDIA knowledge chunks and preserve citation traceability.

## Recommendation Agent

Map diagnosed gaps to candidate technologies with business and technical rationale.

## Briefing Agent

Generate executive-ready output constrained by evidence quality and clarity.
