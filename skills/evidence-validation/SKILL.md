# Skill: Evidence Validation

## Quando usar
Antes de aceitar qualquer afirmacao sobre startup, tecnologia, cliente, funding ou uso de IA.

## Regras
- Fato confirmado exige fonte forte.
- Inferencia precisa ser marcada como inferencia.
- Hipotese fraca nao pode aparecer como conclusao.
- Se nao houver evidencia, escrever "nao verificado".

## Classificacao de confianca
- Alta: fonte oficial ou multiplas fontes confiaveis
- Media: noticia ou diretorio reconhecido
- Baixa: fonte indireta, antiga ou incompleta
