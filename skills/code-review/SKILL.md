# Skill: Critical Code Review

## Funcao
Encontrar falhas tecnicas e de produto.

## Procurar
- Afirmacao sem fonte
- Recomendacao sem gap
- Output generico
- Falta de criterio
- Falta de schema
- Contradicao com o projeto
- Overengineering
- Falta de proximo passo
- Codigo sem teste
- Funcao com responsabilidade demais

## Saida
1. Problemas
2. Gravidade
3. Correcao
4. Versao corrigida, se possivel
