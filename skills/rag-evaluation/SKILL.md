# Skill: RAG Evaluation

## Quando usar
Depois de implementar ou alterar o RAG NVIDIA.

## Avaliar
- A pergunta recuperou chunks relevantes?
- A resposta esta fiel as fontes?
- A tecnologia NVIDIA recomendada foi recuperada corretamente?
- As citacoes estao corretas?
- Houve trecho irrelevante no contexto?

## Metricas
- Precision@k
- Recall@k
- Faithfulness
- Context relevance
- Citation correctness
