# Development Workspace Quality System

## Visão Geral

Sistema de qualidade para o workspace de desenvolvimento assistido por IA do NVIDIA Startup AI Radar.

## Componentes

- `docs/plans/` — plan artifacts versionados
- `docs/adr/` — Architectural Decision Records
- `docs/contracts/` — contratos de desenvolvimento
- `prompts/` — prompts versionados para fluxo de trabalho
- `AGENTS.md` — regras e contexto do workspace
- `ERROR_LOG.md` — registro de erros
- `EVALS.md` — baseline de testes e critérios de qualidade
- Obsidian vault — notas de decisão, resumos, limitações

## Fluxo de Trabalho

Plan → Plan Artifact → Build → Review Diff → Validation → Closure → Commit
