# Output Validation Examples

Examples for Epic 26.2 Workspace Output Validation Gate.

Use these files as small fixtures when manually checking the validators:

- `valid_action_brief.json` - valid minimal Action Brief payload.
- `invalid_action_brief_missing_section.json` - missing required `Evidence` section.
- `invalid_dashboard.json` - dashboard without a required metric.
- `markdown_with_placeholder.md` - Markdown that should produce a controlled warning.

The examples are intentionally small and do not represent real startups.
