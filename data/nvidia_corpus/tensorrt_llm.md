## Description
TensorRT-LLM is an open-source library that optimizes large language model
inference on NVIDIA GPUs. It uses kernel fusion, quantization, and in-flight
batching to maximize throughput and minimize latency. TensorRT-LLM is an
NVIDIA inference backend for production LLM serving.

## Keywords
LLM, inference, optimization, kernel fusion, quantization, throughput, latency, TensorRT

## Use Cases
- Reducing LLM inference cost through quantization and optimized kernels
- Improving latency for production LLM endpoints
- Serving models efficiently with in-flight batching
- Benchmarking inference performance across GPU configurations
