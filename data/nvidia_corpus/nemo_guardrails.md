# NeMo Guardrails

## Product
NeMo Guardrails

## Gaps Addressed
- agent_governance_gap

## Description
NeMo Guardrails is an open-source toolkit for adding programmable guardrails
to LLM-based conversational systems and AI agents. It enables developers to
define safety policies, content moderation rules, and behavior constraints
using a dedicated guardrails definition language. NeMo Guardrails integrates
with major LLM orchestration frameworks.

## Keywords
guardrails, safety, governance, agent, LLM, moderation, policy, NeMo

## Use Cases
- Preventing LLM agents from executing unauthorized actions
- Enforcing content safety policies in customer-facing chatbots
- Implementing role-based access control for AI agent capabilities
- Auditing and logging agent decisions for compliance
