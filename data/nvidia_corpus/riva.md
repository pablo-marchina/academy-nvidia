## Description
NVIDIA Riva is a GPU-accelerated speech AI SDK that provides real-time
speech-to-text (STT), text-to-speech (TTS), and neural language modeling.
Riva supports multiple languages including Brazilian Portuguese, with
customizable models for domain-specific vocabulary and accents.

## Keywords
speech, STT, TTS, voice, audio, ASR, Riva, Portuguese, real-time

## Use Cases
- Building voice-enabled applications with real-time transcription
- Generating natural-sounding speech from text in Brazilian Portuguese
- Customizing speech models for domain-specific vocabulary (e.g., healthcare)
- Deploying low-latency voice AI pipelines in production
